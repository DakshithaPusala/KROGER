ML Model Overview

Linear Regression

Predicts a continuous outcome (e.g., total spend) as a weighted sum of input features. Simple, interpretable, and fast — but assumes a linear relationship between inputs and output. Best used for CLV estimation when relationships are straightforward.

Random Forest

An ensemble of decision trees that vote on the final prediction. Handles non-linear patterns, mixed data types, and is robust to overfitting. Used here for basket analysis to identify which product features drive purchasing combinations.

Gradient Boosting

Builds trees sequentially, each correcting errors from the previous. Typically achieves the highest accuracy among the three. Used here for churn prediction to identify at-risk customers based on transaction history and loyalty behavior.

Selected for CLV: Random Forest — handles the mixed demographic and transaction features in this dataset without requiring feature scaling, and provides feature importance scores to explain which variables drive customer lifetime value.

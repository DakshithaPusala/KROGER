ML Model Overview

Linear Regression

Predicts a continuous outcome (e.g., total spend) as a weighted sum of input features. Simple, interpretable, and fast — best used for CLV estimation when relationships are straightforward.

Random Forest

An ensemble of decision trees that vote on the final prediction. Handles non-linear patterns and mixed data types. Selected for CLV — handles demographic + transaction features without requiring feature scaling.

Gradient Boosting

Builds trees sequentially, each correcting errors from the previous. Highest accuracy among the three. Used for churn prediction to identify at-risk customers based on transaction history and loyalty behavior.
